# Project Overview

This project focuses on Exploratory Data Analysis (EDA) of a Diwali sales dataset to uncover customer purchasing behavior, demographic patterns, and product-level insights. The analysis is performed using Python in a Jupyter Notebook, with visualizations to support data-driven conclusions.

The goal of this project is to understand who buys more, what they buy, and from where, which can help businesses improve marketing strategies, inventory planning, and customer targeting during festive seasons.

## Tools & Libraries Used

- Python

- Jupyter Notebook

- Pandas

- NumPy

- Matplotlib

- Seaborn

## Dataset Description

The dataset contains customer-level Diwali sales data, including:

- Customer demographics (Gender, Age Group, Marital Status)

- Occupation and State

- Product categories and Product IDs

- Order count and purchase amount

## Steps Followed in the Analysis
1. Importing Required Libraries

All necessary Python libraries for data manipulation and visualization were imported.

2. Data Inspection

Checked dataset shape and column information

Identified missing/null values

3. Data Cleaning

Dropped rows with null values

Renamed columns for better readability

Ensured correct data types for analysis

4. Exploratory Data Analysis (EDA)

Multiple visualizations were created to analyze:

Gender-wise purchasing behavior

Age group distribution

State-wise orders and revenue

Marital status vs spending patterns

Occupation-wise customer distribution

Product category performance

Top-selling products

## Key Insights from the Analysis

### Gender-Based Insights

Female customers form the majority of buyers.

Females contribute more to overall revenue compared to male customers.

Female customers also show higher purchasing power.

### Age Group Analysis

Customers aged 26–35 years are the largest and most active buyer group.

Majority of high-value purchases come from women in the 26–35 age group.

### State-Wise Sales

The highest number of orders and total sales come from:

Uttar Pradesh

Maharashtra

Karnataka

### Marital Status

Married women dominate purchasing activity.

This group shows strong spending capacity, especially during festive sales.

### Occupation Insights

Most buyers belong to the following sectors:

IT

Healthcare

Aviation

### Product Category Performance

Top-selling product categories:

Food

Clothing

Electronics
